# Build
1) `mkdir build`
2) `cd build`
3) `cmake ..`
4) `cmake --build .`
5) `cd ..`
6) `javac Main.java`

# Usage
`java -Djava.library.path=build Main "Madam, I'm Adam"` <br>
where "Madam, I'm Adam" can be any word or phrase
